Function Analysis 101

Detailed lecture on analyzing functions with interactive practice.

1. Understanding Functions

A function $f(x)$ is a rule that assigns exactly one output $y$ for each input $x$. To analyze a function, we focus on several key features:

  • Domain: All $x$ values where $f(x)$ is defined. Example: $f(x) = \frac{1}{x}$ has domain $x \neq 0$.
  • Range: All possible outputs. Example: $f(x) = x^2$ has range $y \ge 0$.
  • Asymptotes: Lines that the graph approaches but never reaches. Can be vertical, horizontal, or oblique.
  • Extremums: Points where the function reaches local maxima or minima. Often found by solving $f'(x) = 0$.

2. Domain and Range

Step-by-step:

  1. Domain: Look for values where the function is undefined (division by zero, square roots of negative numbers, logarithms of non-positive numbers).
  2. Range: Analyze the outputs by looking at the graph or solving inequalities for $y$ in terms of $x$.

Example: $f(x) = \sqrt{x-2}$

  • Domain: $x - 2 \ge 0 \Rightarrow x \ge 2$
  • Range: $y \ge 0$

3. Asymptotes

  • Vertical asymptotes: Set denominator = 0. Example: $f(x) = \frac{1}{x-3} \Rightarrow x=3$ is vertical asymptote.
  • Horizontal asymptotes: Compute $\lim_{x\to\infty} f(x)$ and $\lim_{x\to-\infty} f(x)$. Example: $f(x) = \frac{2x+1}{x-3} \Rightarrow y = 2$.
  • Oblique asymptotes: Occur when degree numerator = degree denominator + 1. Use polynomial division.

4. Extrema

Local maxima and minima occur where the derivative is zero or undefined.

Step-by-step:

  1. Compute $f'(x)$.
  2. Solve $f'(x) = 0$ for critical points.
  3. Check $f''(x)$ or use first derivative test to classify maxima/minima.

Example: $f(x) = x^2 - 4x + 3$

  • $f'(x) = 2x - 4 = 0 \Rightarrow x=2$
  • Check second derivative: $f''(x) = 2 > 0 \Rightarrow$ local minimum at $(2, f(2)) = (2, -1)$

Tips

  • Always check for undefined points first for domain and vertical asymptotes.
  • Use limits to determine horizontal asymptotes.
  • Derivatives are your main tool for extremums.
  • GeoGebra is your visual check — compare analytical vs graphical findings.
