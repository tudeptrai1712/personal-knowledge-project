Applications of Derivatives

Using rates of change to understand the behavior of functions.

1. The First Derivative Test: Increasing, Decreasing, and Extrema

The first derivative, $f'(x)$, tells us the slope of the original function $f(x)$. We can use its sign to determine where the function is increasing or decreasing.

If $f'(x)$ is...Then $f(x)$ is...
Positive ($f'(x) > 0$)Increasing
Negative ($f'(x) < 0$)Decreasing
Zero or UndefinedAt a critical point (candidate for max or min).

Systematic Error: Critical Point vs. Extremum

A critical point ($f'(c)=0$) is not automatically a maximum or minimum. For $f(x)=x^3$, $f'(0)=0$, but $x=0$ is neither. Check the sign change of $f'(x)$ around the critical point.

2. The Second Derivative Test: Concavity

The second derivative, $f''(x)$, tells us the rate of change of the slope and helps determine concavity.

If $f''(x)$ is...Then $f(x)$ is...
Positive ($f''(x) > 0$)Concave Up
Negative ($f''(x) < 0$)Concave Down
Zero or UndefinedCandidate for point of inflection.

Second Derivative and Extrema

  • If $f''(c) > 0$, $f(c)$ is local minimum.
  • If $f''(c) < 0$, $f(c)$ is local maximum.
  • If $f''(c) = 0$, inconclusive. Use First Derivative Test.

3. Optimization: Finding the Best Solution

Optimization finds absolute maximum/minimum values under constraints. Steps:

StepAction
1. ModelIdentify objective function and constraints.
2. SubstitutionRewrite objective in single variable using constraints.
3. DomainDetermine feasible domain.
4. CalculusFind derivative and critical points.
5. VerificationEvaluate function at critical points and endpoints.

4. A Worked Example

Analyze $f(x) = x^3 - 6x^2 + 9x + 1$.

ComponentResult
First Derivative$f'(x) = 3x^2 - 12x + 9$
Critical Points$x=1, x=3$
Increasing$(-\infty,1) \cup (3,\infty)$
Decreasing$(1,3)$
Local Maximum$f(1)=5$
Local Minimum$f(3)=1$
Second Derivative$f''(x) = 6x - 12$
Inflection Point$x=2, f(2)=3$
Concave Up$(2,\infty)$
Concave Down$(-\infty,2)$
