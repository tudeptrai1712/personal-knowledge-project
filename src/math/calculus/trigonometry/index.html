Trigonometry for Calculus

A review of the essential trigonometric concepts for calculus.

The Unit Circle and Radians

A strong foundation in trigonometry begins with the unit circle, a circle with radius 1 centered at the origin. For any point $(x, y)$ on the unit circle, the angle $\theta$ (measured in radians) determines the values $\cos(\theta) = x$ and $\sin(\theta) = y$. Using radians is standard in calculus.

Key Trigonometric Identities

Identities are crucial for simplifying expressions before differentiation or integration.

  • Pythagorean Identity: $\sin^2(x) + \cos^2(x) = 1$. Variants include $1 + \tan^2(x) = \sec^2(x)$ and $1 + \cot^2(x) = \csc^2(x)$.
  • Double Angle Identities: These are very useful in integration.
    • $\sin(2x) = 2\sin(x)\cos(x)$
    • $\cos(2x) = \cos^2(x) - \sin^2(x) = 2\cos^2(x) - 1 = 1 - 2\sin^2(x)$

Derivatives of Trigonometric Functions

The derivatives of the six basic trigonometric functions are fundamental to calculus:

  • $\frac{d}{dx}(\sin x) = \cos x$
  • $\frac{d}{dx}(\cos x) = -\sin x$
  • $\frac{d}{dx}(\tan x) = \sec^2 x$
  • $\frac{d}{dx}(\csc x) = -\csc x \cot x$
  • $\frac{d}{dx}(\sec x) = \sec x \tan x$
  • $\frac{d}{dx}(\cot x) = -\csc^2 x$

Integrals of Trigonometric Functions

Reversing the derivative rules gives us the basic integrals:

  • $\int \cos x \,dx = \sin x + C$
  • $\int \sin x \,dx = -\cos x + C$
  • $\int \sec^2 x \,dx = \tan x + C$

More complex integrals often require using identities. For example, to integrate $\cos^2(x)$, we use the identity $\cos^2(x) = \frac{1 + \cos(2x)}{2}$.

Inverse Trigonometric Functions

Inverse trigonometric functions "undo" the trigonometric functions, but with restricted domains. For example, $y = \arcsin(x)$ gives the angle whose sine is $x$. Their derivatives are important algebraic functions:

  • $\frac{d}{dx}(\arcsin u) = \frac{1}{\sqrt{1-u^2}}\frac{du}{dx}$
  • $\frac{d}{dx}(\arccos u) = -\frac{1}{\sqrt{1-u^2}}\frac{du}{dx}$
  • $\frac{d}{dx}(\arctan u) = \frac{1}{1+u^2}\frac{du}{dx}$
