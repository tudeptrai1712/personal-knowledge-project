Antiderivatives

Beyond Reversing Rules: The 'Why' of Finding the Original Function

1. The Core Idea: Reversing a Rate of Change

Concept Description
Differentiation The process of finding the instantaneous rate of change (the slope) of a function. Given $F(x)$, find $F'(x) = f(x)$.
Antidifferentiation The process of reconstructing the original function from its rate of change. Given $f(x)$, find a function $F(x)$ such that $F'(x) = f(x)$.

2. The "Lost Constant" and the Family of Functions

When we differentiate, information about constant terms is lost. This leads to the idea of a "family" of antiderivatives.

Function $F(x)$ Its Derivative $F'(x)$
$x^2$$2x$
$x^2 + 3$$2x$
$x^2 - 100$$2x$
$x^2 + C$ (for any constant C)$2x$

The indefinite integral is not a single function, but an entire family of functions: $$ \int f(x) \,dx = F(x) + C $$

Systematic Error: Forgetting the Constant of Integration

Common MistakeCorrect Understanding
Writing $\int 2x \,dx = x^2$. Writing $\int 2x \,dx = x^2 + C$. You are not finding one function, but the entire family of functions whose derivative is $2x$.

2. F_2(x) = x^2 - 1
3. F_3(x) = x^2
Notice how they are all vertical shifts of each other. Now, see their common derivative by plotting: f(x) = F_1'(x).

3. Systematizing the Rules

Antiderivative rules are simply differentiation rules in reverse.

Differentiation Rule Antiderivative Rule (Indefinite Integral)
$\frac{d}{dx}(x^{n+1}) = (n+1)x^n$ $\int x^n \,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$
$\frac{d}{dx}(\ln x) = \frac{1}{x}$ $\int \frac{1}{x} \,dx = \ln|x| + C$
$\frac{d}{dx}(e^x) = e^x$ $\int e^x \,dx = e^x + C$
$\frac{d}{dx}(\sin x) = \cos x$ $\int \cos x \,dx = \sin x + C$
$\frac{d}{dx}(-\cos x) = \sin x$ $\int \sin x \,dx = -\cos x + C$

Systematic Error: The Absolute Value in $\ln|x|$

FunctionDomain
$f(x) = 1/x$$x \neq 0$
$\ln(x)$$x > 0$ (Domain mismatch!)
$\ln|x|$$x \neq 0$ (Correct domain)

The use of $\ln|x|$ correctly gives us an antiderivative that is defined over the same domain as the original function.

4. A Worked Example: Critical Analysis

Find the indefinite integral of $f(x) = 3x^2 + \cos(x)$.

StepActionResulting Expression
1 Apply Sum Rule: Separate the integral. $\int 3x^2 \,dx + \int \cos(x) \,dx$
2 Apply Constant Multiple Rule. $3 \int x^2 \,dx + \int \cos(x) \,dx$
3 Apply Power and Trig Rules. $3 \left(\frac{x^3}{3}\right) + (\sin(x)) + C$
4 Simplify. $x^3 + \sin(x) + C$

Final Check: The Scientific Method

ComponentFunction / Action
Hypothesis (Our Antiderivative)$F(x) = x^3 + \sin(x) + C$
Verification (Differentiate)$F'(x) = 3x^2 + \cos(x)$
Conclusion$F'(x)$ matches the original integrand $f(x)$. The hypothesis is correct.
