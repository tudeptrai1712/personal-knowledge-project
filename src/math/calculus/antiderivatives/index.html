Antiderivatives

Understanding the reverse process of differentiation.

Definition of an Antiderivative

A function $F$ is called an antiderivative of a function $f$ on an interval $I$ if $F'(x) = f(x)$ for all $x$ in $I$.

For example, if $f(x) = 2x$, then $F(x) = x^2$ is an antiderivative of $f(x)$ because the derivative of $x^2$ is $2x$.

Note that if $F(x)$ is an antiderivative of $f(x)$, then so is $F(x) + C$ for any constant $C$. This is because the derivative of a constant is zero. We call $C$ the constant of integration. The general form of the antiderivative of $f$ is $F(x) + C$.

The Indefinite Integral

The set of all antiderivatives of a function $f(x)$ is called the indefinite integral of $f(x)$ and is denoted by: $$ \int f(x) \,dx = F(x) + C $$ Here, $\int$ is the integral symbol, $f(x)$ is the integrand, and $dx$ indicates that the variable of integration is $x$.

Basic Integration Rules

Just as with differentiation, there are several rules to help us find antiderivatives.

  • Power Rule: $\int x^n \,dx = \frac{x^{n+1}}{n+1} + C$, for $n \neq -1$
  • Constant Rule: $\int k \,dx = kx + C$
  • Constant Multiple Rule: $\int k f(x) \,dx = k \int f(x) \,dx$
  • Sum/Difference Rule: $\int [f(x) \pm g(x)] \,dx = \int f(x) \,dx \pm \int g(x) \,dx$
  • Exponential Functions:
    • $\int e^x \,dx = e^x + C$
    • $\int a^x \,dx = \frac{a^x}{\ln a} + C$
  • Logarithmic Function: $\int \frac{1}{x} \,dx = \ln|x| + C$
  • Trigonometric Functions:
    • $\int \sin(x) \,dx = -\cos(x) + C$
    • $\int \cos(x) \,dx = \sin(x) + C$
    • $\int \sec^2(x) \,dx = \tan(x) + C$

Example

Find the indefinite integral of $f(x) = 3x^2 + 4x - 5$.

Using the rules above, we can integrate term by term: $$ \int (3x^2 + 4x - 5) \,dx = \int 3x^2 \,dx + \int 4x \,dx - \int 5 \,dx $$ $$ = 3 \int x^2 \,dx + 4 \int x \,dx - 5 \int 1 \,dx $$ $$ = 3 \left(\frac{x^3}{3}\right) + 4 \left(\frac{x^2}{2}\right) - 5(x) + C $$ $$ = x^3 + 2x^2 - 5x + C $$ Therefore, the general antiderivative of $f(x) = 3x^2 + 4x - 5$ is $F(x) = x^3 + 2x^2 - 5x + C$.
