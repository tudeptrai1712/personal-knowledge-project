Statistics

Grouped and ungrouped data, core statistical concepts, and references.

Ungrouped Data (Dữ liệu chưa nhóm)

Ungrouped data refers to raw, individual observations that have not been organized into classes or intervals.

  • Mean of Ungrouped Data
  • Median
  • Mode
  • Variance & Standard Deviation

Key Formulas (Ungrouped)

Mean: \[ \bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i \]

Variance: \[ s^2 = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2 \]

Grouped Data (Dữ liệu đã nhóm)

Grouped data organizes observations into class intervals, each with a frequency.

  • Mean of Grouped Data
  • Median for Grouped Data
  • Mode for Grouped Data
  • Variance & SD for Grouped Data
  • Frequency Table Construction

Key Formulas (Grouped)

Mean: \[ \bar{x} = \frac{\sum f_i x_i}{\sum f_i} \]

Class midpoint: \[ x_i = \frac{L_i + U_i}{2} \]

Variance: \[ s^2 = \frac{\sum f_i (x_i - \bar{x})^2}{\sum f_i} \]
