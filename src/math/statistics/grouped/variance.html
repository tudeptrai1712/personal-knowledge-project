Variance & Standard Deviation of Grouped Data

Measuring the spread of data in intervals.

What are Variance and Standard Deviation?

Variance and standard deviation measure how data is spread around the mean. A higher value indicates more spread.

  • Variance ($s^2$): Average squared deviation from the mean.
  • Standard Deviation ($s$): Square root of variance, in original units.

The Formulas

Variance ($s^2$): $s^2 = \frac{\sum f_i (x_i - \bar{x})^2}{\sum f_i - 1}$
Standard Deviation ($s$): $s = \sqrt{s^2}$

  • $f_i$ = Frequency of class
  • $x_i$ = Class midpoint
  • $\bar{x}$ = Mean of grouped data
  • $\sum f_i$ = Total observations

Step-by-Step Example

Using the same student test scores example (mean $\bar{x} = 79.7$).

Class Interval Frequency (f) Class Mark (x) $f_i x_i$
55 - 63359177
64 - 72368204
73 - 81477308
82 - 90586430
91 - 99595475
Total 20 1594
  1. Calculate deviations: $x_i - \bar{x}$
  2. Square deviations: $(x_i - \bar{x})^2$
  3. Multiply by frequency: $f_i (x_i - \bar{x})^2$
  4. Sum values: $\sum f_i (x_i - \bar{x})^2 = 3094.2$
  5. Variance: $s^2 = 3094.2 / (20-1) \approx 162.85$
  6. Standard deviation: $s = \sqrt{162.85} \approx 12.76$

Estimated variance: 162.85, estimated standard deviation: 12.76.
