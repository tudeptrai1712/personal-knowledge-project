Frequency Tables for Grouped Data

Organizing and summarizing large datasets effectively.

What is a Frequency Table?

A frequency table summarizes how often each value or range of values occurs in a dataset. Grouped data uses intervals (classes) to simplify large or continuous datasets.

Key Components of a Grouped Frequency Table

  • Class Intervals: Ranges like 60-69, 70-79.
  • Frequency (f): Count of data points in each class.
  • Relative Frequency: Proportion of total data: $f / n$.
  • Cumulative Frequency: Running total of frequencies.
  • Class Mark (Midpoint): Middle of interval: $(Lower + Upper)/2$.

Constructing a Grouped Frequency Table

  1. Calculate the range: Max - Min.
  2. Decide number of classes $k$ (usually $\approx \sqrt{n}$).
  3. Compute class width: $w = \text{Range}/k$ (round up).
  4. Set class limits: Ensure classes are mutually exclusive and exhaustive.
  5. Tally frequencies.
  6. Optionally, calculate relative and cumulative frequencies.

Example: Student Test Scores

Dataset (20 scores): 55, 60, 63, 67, 70, 72, 75, 78, 80, 81, 83, 85, 87, 88, 90, 92, 95, 96, 97, 98
