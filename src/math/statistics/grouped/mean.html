Mean of Grouped Data

Estimating the average when you only have intervals.

What is the Mean for Grouped Data?

The mean (or average) is a fundamental way to describe the center of a dataset. When you have individual data points, you just add them all up and divide by the count.

However, for grouped data, your data is organized into class intervals (e.g., test scores from 70-79, 80-89) along with how many data points fall into each interval (the frequency). You don't know the exact individual values. In this case, we calculate an estimated mean using a simple formula.

The Formula

The formula to calculate the estimated mean ($\bar{x}$) for grouped data is:

\[ \bar{x} = \frac{\sum f_i x_i}{\sum f_i} \]

Where:

  • $\bar{x}$ = The estimated mean
  • $\sum$ = Sum (meaning "add everything up")
  • $f_i$ = The frequency of each class (how many data points are in that interval)
  • $x_i$ = The class mark (or midpoint) of each class interval

How to Calculate the Mean for Grouped Data (Step-by-Step)

Let's use an example to walk through the process.

Suppose you have the following frequency distribution for student test scores:

Class Interval Frequency (f)
55 - 633
64 - 723
73 - 814
82 - 905
91 - 995
  1. Find the Class Mark ($x_i$) for each Class Interval: The class mark is the midpoint of each interval. Add the lower and upper limits and divide by 2.
    • 55 - 63: $(55+63)/2 = 59$
    • 64 - 72: $(64+72)/2 = 68$
    • 73 - 81: $(73+81)/2 = 77$
    • 82 - 90: $(82+90)/2 = 86$
    • 91 - 99: $(91+99)/2 = 95$
  2. Multiply the Frequency ($f_i$) by the Class Mark ($x_i$) for each class ($f_i x_i$):
    • $3 \times 59 = 177$
    • $3 \times 68 = 204$
    • $4 \times 77 = 308$
    • $5 \times 86 = 430$
    • $5 \times 95 = 475$
  3. Sum the $f_i x_i$ column ($\sum f_i x_i$):

    $\sum f_i x_i = 177 + 204 + 308 + 430 + 475 = 1594$

  4. Sum the Frequencies ($\sum f_i$): This is the total number of data points.

    $\sum f_i = 3 + 3 + 4 + 5 + 5 = 20$

  5. Apply the Formula:

    $\bar{x} = \frac{\sum f_i x_i}{\sum f_i} = \frac{1594}{20} = 79.7$

So, the estimated mean test score for this grouped data is 79.7.

Why is it an "Estimated" Mean?

It's important to remember that this is an estimate. Because we don't know the exact individual scores within each interval, we assume that the midpoint (class mark) is a good representation of all the data in that interval. This gives us a very good approximation of the true mean.
