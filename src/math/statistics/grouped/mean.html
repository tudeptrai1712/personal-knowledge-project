Mean of Grouped Data

Estimating the average using class intervals.

What is the Mean for Grouped Data?

The mean describes the center of a dataset. For individual data points, you simply sum all values and divide by the total count.

For grouped data, values are grouped into intervals with corresponding frequencies. The exact values inside each interval are unknown, so we estimate the mean using class midpoints.

Formula

The estimated mean ($\bar{x}$) is calculated as:

\[ \bar{x} = \frac{\sum f_i x_i}{\sum f_i} \]

  • $f_i$ = Frequency of each class
  • $x_i$ = Class midpoint (or class mark)
  • $\sum$ = Sum across all classes

Step-by-Step Example

Given the following frequency table for student test scores:

Class Interval Frequency (f)
55 - 633
64 - 723
73 - 814
82 - 905
91 - 995
  1. Find Class Midpoints ($x_i$):
    • 55 - 63: $x = (55+63)/2 = 59$
    • 64 - 72: $x = 68$
    • 73 - 81: $x = 77$
    • 82 - 90: $x = 86$
    • 91 - 99: $x = 95$
  2. Multiply Frequency by Midpoint ($f_i x_i$):
    • $3 \cdot 59 = 177$
    • $3 \cdot 68 = 204$
    • $4 \cdot 77 = 308$
    • $5 \cdot 86 = 430$
    • $5 \cdot 95 = 475$
  3. Sum $f_i x_i$: $177 + 204 + 308 + 430 + 475 = 1594$
  4. Sum Frequencies ($\sum f_i$): $3 + 3 + 4 + 5 + 5 = 20$
  5. Calculate Mean: \[ \bar{x} = \frac{1594}{20} = 79.7 \]

The estimated mean score is 79.7.

Why It’s an Estimate

Since we assume that all values in a class interval are represented by the midpoint, the mean is an approximation. It’s usually very close to the true mean for large datasets.
