Mode of Grouped Data

Finding the most frequent interval.

What is the Mode for Grouped Data?

The mode is the value that appears most often in a dataset. For simple, ungrouped data, you just count which number shows up the most.

For grouped data, we don't have individual values. Instead, we have class intervals and their frequencies. We can identify a modal class (the class with the highest frequency). To get a more precise estimate of the mode within that class, we use a specific formula.

The Formula

The formula to calculate the estimated mode for grouped data is:

\[ \text{Mode} = L + \left( \frac{f_m - f_1}{(f_m - f_1) + (f_m - f_2)} \right) w \]

Where:

  • $L$ = The lower boundary of the modal class.
  • $f_m$ = The frequency of the modal class.
  • $f_1$ = The frequency of the class *before* the modal class.
  • $f_2$ = The frequency of the class *after* the modal class.
  • $w$ = The width of the modal class.

How to Calculate the Mode for Grouped Data (Step-by-Step)

Let's use an example to walk through the process.

Suppose you have the following frequency distribution for student test scores:

Class Interval Frequency (f)
55 - 633
64 - 723
73 - 814
82 - 905
91 - 995
  1. Identify the Modal Class: This is the class with the highest frequency. In our table, both 82 - 90 and 91 - 99 have a frequency of 5. When there are two classes with the same highest frequency, the distribution is bimodal. For this example, let's pick 82 - 90 as our modal class.
  2. Identify the values for the formula ($L, f_m, f_1, f_2, w$):
    • $L$ = Lower boundary of the modal class = 82 - 0.5 = 81.5
    • $f_m$ = Frequency of the modal class (82 - 90) = 5
    • $f_1$ = Frequency of the class before the modal class (73 - 81) = 4
    • $f_2$ = Frequency of the class after the modal class (91 - 99) = 5
    • $w$ = Class width = 9
  3. Apply the Formula:

    Mode = $81.5 + \left( \frac{5 - 4}{(5 - 4) + (5 - 5)} \right) 9$

    Mode = $81.5 + (1) 9$

    Mode = $90.5$

So, the estimated mode test score for this grouped data is 90.5.

Important Note on Multimodal Distributions

Sometimes, a grouped frequency distribution might have two or more classes with the highest frequency. In such cases, the distribution is considered bimodal (two modes) or multimodal (multiple modes). Our formula gives the mode for a single modal class.
