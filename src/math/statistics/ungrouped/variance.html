Variance & Standard Deviation of Ungrouped Data

Measuring the spread of raw (ungrouped) data values.

What Are We Measuring?

For ungrouped data (a simple list of numbers), the idea is exactly the same as for grouped data:

  • Variance ($s^2$): The average of the squared deviations from the mean.
  • Standard Deviation ($s$): The square root of the variance.

These values tell you how spread out your dataset is. A small standard deviation means the numbers stick close to the mean, while a large one means they vary widely.

The Formulas

Sample Variance:

\[ s^2 = \frac{\sum (x_i - \bar{x})^2}{n - 1} \]

Standard Deviation:

\[ s = \sqrt{s^2} \]

Where:

  • $x_i$ = each individual data value
  • $\bar{x}$ = the mean of the data
  • $n$ = number of observations
  • $n - 1$ is used for sample data (most common in statistics classes)

Example: Ungrouped Test Scores

Suppose we have the following test scores from 10 students:

$72,\ 85,\ 90,\ 68,\ 88,\ 79,\ 94,\ 81,\ 76,\ 73$

  1. Find the Mean ($\bar{x}$):

    Add up all values:

    $72 + 85 + 90 + 68 + 88 + 79 + 94 + 81 + 76 + 73 = 806$

    Divide by number of data points ($n = 10$):

    $\bar{x} = \frac{806}{10} = 80.6$

  2. Compute Each Deviation ($x_i - \bar{x}$):
    • $72 - 80.6 = -8.6$
    • $85 - 80.6 = 4.4$
    • $90 - 80.6 = 9.4$
    • $68 - 80.6 = -12.6$
    • $88 - 80.6 = 7.4$
    • $79 - 80.6 = -1.6$
    • $94 - 80.6 = 13.4$
    • $81 - 80.6 = 0.4$
    • $76 - 80.6 = -4.6$
    • $73 - 80.6 = -7.6$
  3. Square Each Deviation:
    • $(-8.6)^2 = 73.96$
    • $(4.4)^2 = 19.36$
    • $(9.4)^2 = 88.36$
    • $(-12.6)^2 = 158.76$
    • $(7.4)^2 = 54.76$
    • $(-1.6)^2 = 2.56$
    • $(13.4)^2 = 179.56$
    • $(0.4)^2 = 0.16$
    • $(-4.6)^2 = 21.16$
    • $(-7.6)^2 = 57.76$
  4. Add All Squared Deviations:

    $\sum (x_i - \bar{x})^2 = 656.4$

  5. Compute Sample Variance:

    $s^2 = \frac{656.4}{10 - 1} = \frac{656.4}{9} \approx 72.93$

  6. Compute Standard Deviation:

    $s = \sqrt{72.93} \approx 8.54$

So the sample variance of these test scores is approximately 72.93, and the standard deviation is about 8.54. This means, on average, the scores differ from the mean by around 8.5 points.
