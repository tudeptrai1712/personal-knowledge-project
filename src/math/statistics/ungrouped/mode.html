Mode of Ungrouped Data

Finding the most frequent value in a simple dataset.

What is the Mode?

The mode is the value that occurs most frequently in a dataset.

It is the simplest measure of central tendency and is especially useful for identifying the most common value or category in a set of data.

Unlike the mean or median, the mode:

  • Can be used for numerical or categorical (non-numerical) data
  • May have one mode (unimodal), more than one (multimodal), or no mode at all

How to Find the Mode

The steps are straightforward:

  1. List the data.
  2. Count how many times each value appears.
  3. The value with the highest frequency is the mode.

Example

Consider the following set of student quiz scores:

\[ 7,\; 9,\; 9,\; 10,\; 6,\; 7,\; 9,\; 8,\; 6,\; 7 \]

Let's count each value:

Value Frequency
62
73
81
93
101

The values 7 and 9 both appear 3 times — the highest frequency.

This dataset is bimodal: the modes are 7 and 9.

Important Notes

  • If all values occur the same number of times, the dataset has no mode.
  • If more than one value ties for highest frequency, the distribution is multimodal.
  • The mode is the only central tendency measure appropriate for categorical data (e.g., colors, names, brands).
