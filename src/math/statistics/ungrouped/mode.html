Mode of Ungrouped Data

Finding the most frequent value in a simple dataset.

Definition

The mode is the value that appears most frequently in a dataset.

It can be used for both numerical and categorical data, unlike the mean or median.

Steps to Find the Mode

  1. List all data points.
  2. Count the frequency of each value.
  3. The value(s) with the highest frequency is/are the mode.

Example

Consider the dataset:

$7, 9, 9, 10, 6, 7, 9, 8, 6, 7$

ValueFrequency
62
73
81
93
101

This dataset has two values with the highest frequency: 7 and 9.

The dataset is bimodal.

Notes

  • If all values occur the same number of times, there is no mode.
  • If multiple values tie for highest frequency, the dataset is multimodal.
  • The mode is the only measure of central tendency that can be used for categorical data.
