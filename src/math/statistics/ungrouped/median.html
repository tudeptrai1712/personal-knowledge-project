Median of Ungrouped Data

Finding the middle value of a dataset.

What is the Mode?

The mode is the value that occurs most frequently in a dataset.

It is useful for identifying the most common value or category. Unlike mean or median, the mode:

  • Can be used for numerical or categorical data
  • May have one mode (unimodal), more than one (multimodal), or no mode

How to Find the Mode

  1. List the data.
  2. Count how many times each value appears.
  3. The value(s) with the highest frequency is the mode.

Example

Consider the student quiz scores:

\[ 7, 9, 9, 10, 6, 7, 9, 8, 6, 7 \]

Value Frequency
62
73
81
93
101

This dataset is bimodal: the modes are 7 and 9.

What is the Median?

The median is the middle value of a dataset when the numbers are arranged in order.

It divides the data into two equal halves — 50% of values are below it, and 50% are above it.

The median is:

  • Unaffected by extreme values (outliers)
  • More representative than the mean for skewed data
  • Always based on ordered values

How to Find the Median

Follow these steps:

  1. Arrange the data in ascending order.
  2. Count the total number of observations \( n \).
  3. Determine whether \( n \) is **odd** or **even**:
    • If \( n \) is odd: the median is the middle value.
    • If \( n \) is even: the median is the average of the two middle values.

Median Formulas

Case 1: Odd number of values

\[ Me = x_{\frac{n+1}{2}} \]

Case 2: Even number of values

\[ Me = \frac{x_{\frac{n}{2}} + x_{\frac{n}{2}+1}}{2} \]

Example

Dataset:

\[ 12,\; 7,\; 15,\; 10,\; 9,\; 8 \]

Step 1: Order the data

\[ 7,\; 8,\; 9,\; 10,\; 12,\; 15 \]

Step 2: Count the values

There are \( n = 6 \) values (even).

Step 3: Find the median

The two middle values are:

\[ x_3 = 9,\quad x_4 = 10 \]

Apply the formula:

\[ Me = \frac{9 + 10}{2} = 9.5 \]

The median of this dataset is 9.5.

Important Notes

  • If all values occur the same number of times, there is no mode.
  • If more than one value ties for highest frequency, the distribution is multimodal.
  • The mode is appropriate for categorical data (e.g., colors, names, brands).
  • The data must be sorted before finding the median.
  • For datasets with repeated values, the procedure is unchanged — only the position matters.
  • The median is a robust measure and does not get distorted by extremely large or small values.
