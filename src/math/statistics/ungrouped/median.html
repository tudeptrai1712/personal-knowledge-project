Mode of Ungrouped Data

Finding the most frequent value in a simple dataset.

What is the Mode?

The mode is the value that occurs most frequently in a dataset.

It is useful for identifying the most common value or category. Unlike mean or median, the mode:

  • Can be used for numerical or categorical data
  • May have one mode (unimodal), more than one (multimodal), or no mode

How to Find the Mode

  1. List the data.
  2. Count how many times each value appears.
  3. The value(s) with the highest frequency is the mode.

Example

Consider the student quiz scores:

\[ 7, 9, 9, 10, 6, 7, 9, 8, 6, 7 \]

Value Frequency
62
73
81
93
101

This dataset is bimodal: the modes are 7 and 9.

Important Notes

  • If all values occur the same number of times, there is no mode.
  • If more than one value ties for highest frequency, the distribution is multimodal.
  • The mode is appropriate for categorical data (e.g., colors, names, brands).
